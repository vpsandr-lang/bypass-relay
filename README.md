# 🏢 Виртуальный офис-менеджер для нотариальной конторы

Интерактивная 3D система с реалистичным аватаром для консультирования посетителей, 
ответов на вопросы и записи на приём к нотариусу.

## ✨ Возможности

- **3D аватар** — реалистичная модель человека (по пояс) с анимацией речи, моргания, дыхания
- **🎤 Голосовое общение** — распознавание речи и синтез ответов (русский язык)
- **📹 Веб-камера** — отображение посетителя на экране
- **🤖 Интеллектуальные ответы** — на базе Ollama (qwen2.5) с fallback-системой
- **📅 Запись на приём** — модальная форма с сохранением в базу данных
- **🔊 Озвучивание ответов** — реалистичная русская речь с синхронизацией губ

## 🚀 Быстрый старт

```bash
# 1. Установка
chmod +x setup.sh
./setup.sh

# 2. Запуск
node server.js

# 3. Открыть в браузере
# http://localhost:3000
```

## 📋 Требования

- Node.js 18+
- Ollama (устанавливается автоматически)
- Современный браузер (Chrome, Yandex Browser для голосового ввода)
- Веб-камера (опционально)

## 🎭 3D Аватар

Система поддерживает два режима отображения:

1. **VRM модель** — загружается из файла `public/models/avatar.vrm`
   - Поддерживаются модели формата VRM (Vroid, Ready Player Me)
   - Реалистичная анимация лица, губ, моргание
   
2. **Процедурный аватар** — запасной вариант на Three.js
   - Создаётся автоматически если VRM модель недоступна
   - Базовая анимация: дыхание, моргание, движение губ

### Установка своей VRM модели

1. Создайте аватар на https://vroid.com или https://readyplayer.me
2. Скачайте VRM файл
3. Поместите в `public/models/avatar.vrm`
4. Перезагрузите страницу

## 🤖 Настройка LLM

По умолчанию используется модель `qwen2.5:3b` через Ollama.
Для смены модели:

```bash
ollama pull qwen2.5:7b   # Более качественная модель
export LLM_MODEL=qwen2.5:7b
node server.js
```

## 🔌 API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/health` | Проверка статуса |
| POST | `/api/chat` | Отправка сообщения |
| GET | `/api/appointments` | Список записей |
| POST | `/api/appointments` | Новая запись |
| DELETE | `/api/appointments/:id` | Отмена записи |

## 🌐 Браузерная поддержка

- **Chrome** — полная поддержка (речь, 3D, камера)
- **Yandex Browser** — полная поддержка (лучшее распознавание русского)
- **Firefox** — 3D и камера работают, речь может быть ограничена
- **Safari** — базовая поддержка

## 📞 Контакты

Для настройки под конкретную нотариальную контору измените:
- `SYSTEM_PROMPT` в `server.js` — описание услуг и режим работы
- `public/index.html` — название конторы в шапке
- `public/css/style.css` — цветовую схему
